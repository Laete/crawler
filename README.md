# MVP Crawler

## Requirements
- python 3.8.*
- pip3

## Installation
```shell
git clone https://github.com/Laete/crawler.git
python3 -m venv .venv
source .venv/bin/activate
pip3 install
```

## Run
```shell
python3 manage.py migrate
python3 manage.py runserver
```
